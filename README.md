# nhentai-downloader
a rewrite of the nhentai-downloader in full C  

## installation
```
$ git clone https://github.com/hrlou/nhentai_c
$ cd nhentai_c
$ sudo make install
```

## configuration
Too edit the configuration, open "config.def.h"  
Instructions are included in there
