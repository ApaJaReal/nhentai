# nhentai-downloader
a rewrite of the nhentai-downloader in full C  
there will be **no config file** for this version, options will be done through a header file.

## installation
```
$ git clone https://github.com/hrlou/nhentai_c
$ cd nhentai_c
$ sudo make install
```
